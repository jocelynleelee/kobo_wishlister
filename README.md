# discount_alert
create a wishlist for books found in Kobo (TW)

- GUI:
  * register
  * login
  * add a book (using book id) to your wishlist
  * check your wishlist


- supported APIs:
  * submit_task(book_id)
  * update_wishlist()
  * get_wishlist()


- modules used:
  * flask
  * SQLAlchemy
